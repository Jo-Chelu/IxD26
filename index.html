STARS Interaction Design Report

Student: Jo Chelu

Introduction

This report analyses key user tasks within the STARS Interaction Design system available at compucore3.setu.ie/stars_ixd. The aim of this report is to understand how students interact with the platform while completing common academic tasks.

Three user workflows were analysed: creating a brief, submitting an assignment, and adding a note for reviewers. These tasks represent common interactions where students submit work and communicate information to teachers through the system.

Each flow was completed multiple times to observe user actions, system responses, and possible usability issues. Any confusion, hesitation, or errors were recorded as pain points. These findings will later help inform design improvements and prototype solutions in the next stages of the project.
